这个项目是一个“山东省高考志愿投档录取核对系统”。它是一个纯前端的单文件工具（HTML+JS），专门为高考考生量身定制，用于在志愿填报后进行模拟投档预测和往年录取数据检索。

它的核心功能和特点可以概括为以下三点：

1. 🤖 志愿表 PDF 全自动比对（核心亮点）
   
智能解析：考生只需输入自己的高考精准位次，并上传官方导出的志愿填报 PDF 文件。

模拟平行志愿投档：系统会在本地自动解析 PDF 中的院校代码和专业代码，并依次对比数据库。

精准定位录取点：严格模拟平行志愿“分数优先、遵循志愿”的投档规则。一旦发现某个志愿过线，会将其高亮标记为 🏆 最终录取，并自动将后续志愿拦截（标记为“已被前志愿截断”），直接让考生看到自己最终可能滑落到第几个志愿。

2. 🔍 多年份数据智能检索
   
跨表模糊查询：系统支持手动精确查找。输入院校代号/名称、专业代号/名称或位次，即可一键横跨数据库内的多个年份（如 2024、2025、2026 年）进行对比。

动态过线判定：如果输入了考生位次，检索结果会自动根据往年投档线，直观地用红绿标签呈现“过线”或“未过线”。

3. 🔒 100% 本地运行与隐私保护
   
安全无上传：系统利用了 WebAssembly (sql-wasm.js) 技术，在浏览器本地直接运行和查询 SQL 数据库（.db 文件），同时在本地解析 PDF。

零服务器依赖：考生的成绩、位次和志愿表等极度敏感的隐私数据绝不上传到任何服务器，完全在用户自己的电脑浏览器内安全完成计算。

一键导出：比对完成后，支持将完整的模拟投档清单一键导出为本地 CSV 表格，方便留存和分析。
